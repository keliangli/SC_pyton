## 关键信号
- GGUF 是当前 ModelScope 上最主流的量化模型发布形态，接近一半量化模型归到该类。
- 显式以 int3 / int4 / int8 命名的模型很多，说明社区存在大量“直接以位宽命名”的发布习惯，而不一定总是注明 AWQ/GPTQ。
- AWQ 与 GPTQ 规模接近，AWQ 略高于 GPTQ。
- EXL2、HQQ、AQLM、EETQ 等更偏长尾生态。
- 这份统计是“平台模型命名/标签分布”，不是官方 benchmark 分布。

## 统计口径说明
- 数据源：ModelScope 官方列表接口 `api/v1/models`
- 分类字段：Name / ChineseName / Tags / OfficialTags / Libraries / Frameworks / ModelType / Description
- 互斥主类别：每个模型只归入一个最高优先级量化类别
- 多标签统计：原始明细见 `modelscope_quant_type_distribution_2026-03-09.json`
